# 画像つきの掲示板

## 機能
- ユーザーごとのログイン機能
- 画像つきの投稿ができる
- フォローしているユーザーの投稿を閲覧
- フォロー/フォロー解除
- 自分の投稿の編集・削除

## 目的
- Slimに慣れる
- PHPでの画像の扱い方を知る
- Travis CIを使ってみる
